\# Лабораторная работа №9



\## Ход лабораторной работы



1\. Создана папка лабораторной работы и инициализирован локальный репозиторий с помощью команды `git init`.



2\. Выполнена настройка имени пользователя и адреса электронной почты с использованием команды `git config`.



3\. Проверено текущее состояние репозитория с помощью команды `git status`.



4\. Создан файл `README.md`, добавлен в индекс и зафиксирован первым коммитом.



5\. Создан удалённый репозиторий на GitHub и выполнена привязка локального репозитория к удалённому.



6\. Выполнена отправка локальных коммитов в удалённый репозиторий командой `git push`.



7\. Создана новая ветка `mybranch` для работы с ветвлением.



8\. Выполнено переключение на ветку `mybranch` и проверено её состояние.



9\. В ветке `mybranch` создан файл `file1.txt`, добавлен в репозиторий и закоммичен.



10\. Просмотрена история коммитов текущей ветки с помощью команды `git log`.



11\. Выполнен переход обратно в основную ветку `master`.



12\. В ветке `master` создан файл `file2.txt` и выполнен коммит.



13\. Просмотрена история коммитов всех веток с помощью команды `git log --all`.



14\. Проверено изменение файлов при переключении между ветками.



15\. Выполнено сравнение веток `master` и `mybranch` с помощью команды `git diff`.



16\. Добавлена папка `screenshots` со скриншотами выполнения лабораторной работы и отправлена в удалённый репозиторий.



17\. Проверено наличие всех файлов и веток в удалённом репозитории на GitHub.
